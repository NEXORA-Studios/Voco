# Voco

## 🎯 产品概述

Voco 是一款专为教育场景设计的桌面应用，帮助老师轻松管理词汇包并高效提问学生。通过直观的界面和灵活的功能，老师可以快速创建、导入和管理词汇包，并在课堂上或远程教学中方便地使用这些词汇包进行互动提问。

## ✨ 核心功能

### 👨‍🏫 教师专属功能

-   **词汇包管理**：创建、编辑、删除和组织词汇包
-   **批量导入**：支持从多种格式导入词汇包数据
-   **灵活提问**：随机或按顺序抽取词汇进行提问
-   **课堂模式**：适合现场教学的全屏展示模式

### 📦 词汇包系统

-   支持 CSV 等多种格式导入
-   每个词汇包可包含词汇、释义、例句等完整信息
-   支持分类和标签管理，便于快速查找
-   轻松导出词汇包，方便分享和备份

### 🎮 交互式提问

-   简洁直观的提问界面，减少操作干扰
-   可自行调整显示时间和回答模式

### 🌍 多语言支持

-   支持简体中文、繁体中文、英语（美国）、英语（英国）
-   界面自动适配系统语言

### 💾 本地数据安全

-   所有词汇包和记录保存在本地设备
-   支持数据备份和恢复
-   保护教学资料隐私安全

## 🚀 快速开始

### 系统要求

-   Windows 10/11 (64-bit)
-   macOS 10.15 或更高版本
-   Linux (基于 Debian/Ubuntu 或 Fedora)

### 下载安装

1. 从 [GitHub Releases](https://github.com/NEXORA-Studios/Voco/releases) 下载对应平台的安装包
2. 运行安装程序，按照提示完成安装
3. 启动 Voco 应用

## 📖 老师使用指南

### 主界面介绍

-   **词汇包管理区**：显示所有词汇包，支持搜索和筛选
-   **工具栏**：提供新建、导入、导出等操作按钮
-   **设置面板**：调整应用和提问设置
-   **统计视图**：查看学生表现和练习记录

### 词汇包管理

#### 创建词汇包

1. 点击 "新建词汇包" 按钮
2. 输入词汇包名称和描述
3. 选择分类和语言
4. 点击 "创建" 按钮

#### 导入词汇包

1. 点击 "导入词汇包" 按钮
2. 选择词汇包文件（目前仅支持 CSV 格式）
3. 按照提示填写词汇集和词汇包名
4. 点击 "导入" 按钮

### 提问流程

#### 准备提问

1. 选择要使用的词汇包
2. 点击右侧按钮加载

#### 进行提问

1. 点击 "下一个" 开始随机词汇
2. 点击 "停止" 展示正确答案
3. 提问学生之后点击 "显示中文" 展示正确答案
4. 点击 "下一个" 继续提问

#### 词汇包提问完成后

-   点击 "返回首页" 按钮返回主界面
-   点击 "重新开始" 重新开始整个词汇包

-   简洁直观的界面，减少教学过程中的操作干扰
-   清晰的视觉层次，突出核心内容
-   流畅的交互体验，提升教学效率
-   响应式设计，适配不同教学环境

## 🤝 社区与支持

**Q: 词汇包数据保存在哪里？**
A: 所有词汇包和记录保存在本地设备上，确保教学资料的隐私安全。

A: 目前仅支持 CSV 格式导入词汇包。

**Q: 是否支持离线使用？**
A: 是的，Voco 完全支持离线使用，无需网络连接。
